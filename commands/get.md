# Command Definitions & Usage

## get

- Definition
  - Gets files from the internet.
- Usage
  - `get [url] [file name]`
  - Replace `[url]` with the URL of the file you're downloading.
  - Replace `[file name]` with the name of the file you want to download to.
- Aliases
  - `get`
  - `download`
