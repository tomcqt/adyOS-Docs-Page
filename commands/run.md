# Command Definitions & Usage

## run

- Definition
  - Lets you run your own JavaScript files within adyOS.
- Usage
  - `run [file name]`
  - Replace `[file name]` with the name of the JavaScript file that you want to run.
- Aliases
  - `run`
  - `open`
  - `start`
