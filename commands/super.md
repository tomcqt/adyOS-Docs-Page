# Command Definitions & Usage

## super

- Definition
  - Basically just `sudo` from Linux.
  - Requires special permissions.
- Usage
  - `super [command]`
  - Replace `[command]` with the command that you want to run.
- Aliases
  - `super`
  - `sudo`
